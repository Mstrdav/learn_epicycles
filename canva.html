Votre formule

Ce site intéractif vous permet de dessiner des formes qui seront analysées avec la transformée de Fourier pour les redessiner à l'aide de cercle de taille et de vitesse variable.

    Voici quelques-unes des fonctionnalités :
  • Recentrer le dessin. Un bouton dans le menu de droite permet de recentrer le dessin. Cliquez dessus pour que le centre de gravité de votre dessin se retrouve au centre.
  • Dessiner une forme. A l'aide de votre souris, dessinez une forme fermée, l'ordinateur calculera les cercles correspondant. Si vous ne la fermez pas, l'ordinateur la fermera lui-même par une dernière ligne droite.
  • Charger des modèles. Via le menu sur la droite, vous pouvez ouvrir une bibliothèque de modèles. Cliquez ensuite sur le modèle de votre choix pour le charger.
  • Régler la précision. Utilisez la molette de votre souris pour régler la précision.
  • Régler la vitesse. Utiliser la molette de votre souris, tout en appuyant sur les touches alt ou espace pour régler la vitesse. Plus la vitesse est grande, moins le tracé est fluide.
  • Afficher la formule. Un bouton dans le menu permet d'afficher la formule. Vous pouvez ensuite la copier en cliquant dessus ou sur le petit bouton copier.
